Understanding Semantic HTML Tags

  • Header
  • Nav
  • Main
  • Section
  • Article
  • Aside
  • Footer

Header

The <header> element represents a container for introductory content or a set of navigational links. It typically contains one or more heading elements (<h1> - <h6>), logo, or icons.

Nav

The <nav> element represents a section of a page that links to other pages or to parts within the page. It is intended for major navigational blocks.

Section

The <section> element represents a standalone section of content which is typically accompanied by a heading. It is used to group related content together.

Article

The <article> element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.

Aside

The <aside> element represents a section of a document with content connected tangentially to the content around the aside element (often presented as a sidebar).

Footer

The <footer> element represents a footer for its nearest sectioning content or sectioning root element. It typically contains information about the author, copyright information, or links to related documents.
