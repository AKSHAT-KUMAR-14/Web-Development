HTML Entities Explained

HTML entities are special codes used to display reserved characters in HTML or ensure certain characters are shown correctly. Below are common examples:

Character Entity Name Entity Number Description
& &amp; &#38; Used to represent an ampersand (&) character.
< &lt; &#60; Represents the "less than" symbol.
> &gt; &#62; Represents the "greater than" symbol.
" &quot; &#34; Represents a double quote.
' &apos; &#39; Represents a single quote.
  &nbsp; &#160; Represents a non-breaking space.
© &copy; &#169; Copyright symbol.
® &reg; &#174; Registered trademark symbol.
€ &euro; &#8364; Euro currency symbol.
¥ &yen; &#165; Yen currency symbol.
