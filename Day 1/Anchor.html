Understanding the Anchor Tag (<a>)

The anchor tag is used to create hyperlinks in HTML. It is one of the most important tags in HTML as it allows users to navigate from one page to another.

Basic Syntax

<a href="URL">Link Text</a>

The href attribute specifies the URL of the page the link goes to.

Example

Here is an example of a simple anchor tag:

<a href="https://www.example.com">Visit Example.com</a>

This will create a link that says "Visit Example.com" and will take the user to "https://www.example.com" when clicked.

Opening Links in a New Tab

To open a link in a new tab, use the target attribute with the value _blank:

<a href="https://www.example.com" target="_blank">Visit Example.com</a>

Linking to an Email Address

You can also use the anchor tag to create a link that opens the user's email client:

<a href="mailto:someone@example.com">Send Email</a>

Linking to a Section on the Same Page

To link to a specific section on the same page, use the id attribute:

<a href="#section1">Go to Section 1</a>

Then, add the corresponding id to the target section:

<h2 id="section1">Section 1</h2>
