HTML Lists

HTML provides three types of lists:

  • Ordered List
  • Unordered List
  • Definition List

Ordered List

An ordered list is a list of items in which the order does matter. It is created using the <ol> tag.

  1. First item
  2. Second item
  3. Third item

Here are various types of ordered lists:

  1. First item
  2. Second item
  3. Third item

Ordered list with lowercase letters:

  1. First item
  2. Second item
  3. Third item

Ordered list with uppercase letters:

  1. First item
  2. Second item
  3. Third item

Ordered list with lowercase Roman numerals:

  1. First item
  2. Second item
  3. Third item

Ordered list with uppercase Roman numerals:

  1. First item
  2. Second item
  3. Third item

Unordered List

An unordered list is a list of items in which the order does not matter. It is created using the <ul> tag.

  • First item
  • Second item
  • Third item

Definition List

A definition list is a list of terms and their definitions. It is created using the <dl> tag along with <dt> for the term and <dd> for the definition.
