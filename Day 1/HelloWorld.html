Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language for creating web pages and web applications. HTML describes the structure of a web page using markup.

HTML Elements & Tags

HTML elements are the building blocks of HTML pages. They are represented by tags. Tags are enclosed in angle brackets, and most tags come in pairs: an opening tag and a closing tag.

Paragraph Element

The paragraph element is used to define a block of text. It is represented by the <p></p> tag. For example:

<p>This is a paragraph.</p>
<p>This is a paragraph.</p>

Heading Element

Heading elements are used to define headings in a document. There are six levels of headings, from <h1> to <h6>, with <h1> being the highest level and <h6> the lowest. For example:

<h1>This is a heading</h1>

Boilerplate Code

Boilerplate code refers to the basic HTML structure that is required to start creating a web page. It includes the <!DOCTYPE html> declaration, the <html> element, the <head> element, and the <body> element. Here is an example of boilerplate code:

<!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    </head>
    <body>
    <p>Hello World</p>
    </body>
    </html>

Learning HTML

HTML is easy to learn and widely used for creating web pages.

HTML Maintenance

It is maintained by the World Wide Web Consortium (W3C).

HTML5

HTML5 is the latest version of HTML and includes new features and improvements.

File Extensions

HTML documents are saved with a .html or .htm file extension.

Combining HTML with Other Technologies

HTML can be combined with CSS and JavaScript to create dynamic and interactive web pages.
