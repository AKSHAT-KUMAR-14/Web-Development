Image Tag in HTML

The <img> tag is used to embed images in an HTML page. It is an empty tag, meaning it does not have a closing tag. The <img> tag has two required attributes:

  • src: Specifies the path to the image file.
  • alt: Provides alternative text for the image if it cannot be displayed.

Example

Here is an example of an image tag:

<img src="path/to/image.jpg" alt="Description of the image">
