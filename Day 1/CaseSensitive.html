Is HTML Case Sensitive?

No, HTML is not case sensitive. This means that the HTML tags and attributes can be written in uppercase, lowercase, or a mix of both. For example, the following tags are all equivalent:

  • <html> and <HTML>
  • <body> and <BODY>
  • <h1> and <H1>

However, it is a common practice to write HTML tags and attributes in lowercase to maintain consistency and readability.
